# Zoo Management System

A Java OOP project simulating a zoo using inheritance, interfaces, and polymorphism.  
Animals are stored in a 3x5 2D array and can perform actions such as feeding, cleaning, and making sounds.

## Features
- Abstract Animal class
- Lion, Elephant, and Giraffe subclasses
- Interface for common actions (feed, clean)
- 2D array zoo layout
- Polymorphic behavior
- Simple simulation of zoo operations

## Skills Demonstrated
- Object‑Oriented Programming (OOP)
- Inheritance and abstract classes
- Interfaces and method overriding
- 2D arrays
- Clean, readable Java code
