# Zoo Management System

A Java OOP project simulating a zoo using inheritance, interfaces, and polymorphism. Animals are stored in a 2D array and can perform actions such as feeding, cleaning, and making sounds.

## Features
- Abstract Animal class
- Lion, Elephant, and Giraffe subclasses
- Interface for common actions
- 2D array zoo layout
- Polymorphic behavior

## Skills Demonstrated
- OOP (inheritance, abstract classes, interfaces)
- Polymorphism
- 2D arrays
- Method overriding
